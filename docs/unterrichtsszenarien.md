# Unterrichtsszenarien

## Szenario A: „Beziehungsknoten“ (Matura, 2 Lektionen)
- Phase 1: Board-Anordnung (10 min)
- Phase 2: 3 Profile öffnen, Beziehungskarten sammeln (15 min)
- Phase 3: Jede Gruppe präsentiert 1 Beziehung als Kausal-Kette (15 min)
- Phase 4: Zeitbruch-Übertragung + Diskussion (25 min)
- Phase 5: Schreibauftrag: Mini-Essay (Hausaufgabe)

**Mini-Essay-Prompt:**
„Zeige an einer Beziehung, wie Seghers Privates als politisch beschreibt.“

## Szenario B: Beamer-Plenum (45 min)
- Lehrperson platziert live (Provokation)
- SuS widersprechen / verschieben
- Profilkarten als Textanker

## Szenario C: Prüfungsvorbereitung
- Export-State als Lernstand
- SuS müssen 5 Zuordnungen schriftlich begründen (je 5–6 Sätze)
